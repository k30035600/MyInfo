# 프로젝트 이름·정보 — GitHub 동일하게

아래 값을 **GitHub**에 넣으면 됩니다.

---

## 사용할 이름·설명 (공통)

| 항목 | 값 |
|------|-----|
| **이름** | `MyInfo` |
| **설명(한 줄)** | `은행·신용카드 거래 전처리·분석 통합 (Flask)` |

---

## GitHub에 설정

1. 저장소 페이지 → **Settings**
2. **Repository name**: `MyInfo` (이미 같으면 생략)
3. 저장소 **메인 페이지** → **About** (우측 연필 아이콘) 클릭
4. **Description**에 입력: `은행·신용카드 거래 전처리·분석 통합 (Flask)`
5. **Website**에 배포 URL 넣어도 됨

---

## 저장소가 MyBankCard인 경우

GitHub에서 **MyBankCard** 저장소를 쓰는 경우:

| 항목 | 값 |
|------|-----|
| **이름** | `MyBankCard` |
| **설명** | `은행·신용카드 거래 전처리·분석 통합 (Flask)` |

위와 같이 GitHub **About** / **Repository name**에 같은 값으로 넣으면 됩니다.
