## 3. Railway — 공개 URL(도메인) 생성

**로그인은 완료된 상태입니다.** 터미널(PowerShell/CMD)에서 **연결** 후 도메인만 생성하면 됩니다.

**현재 이 폴더는 Railway 프로젝트에 연결되어 있지 않습니다.** 아래 순서대로 실행하세요.

```powershell
cd d:\OneDrive\Cursor_AI_Project\MyInfo
npx @railway/cli login
npx @railway/cli link
npx @railway/cli domain
```

- `railway login`: 브라우저에서 Railway 로그인 (한 번만)
- `railway link`: 프로젝트·서비스 선택 (대화형, GitHub로 만든 프로젝트 선택)
- `railway domain`: 공개 URL 생성 → 터미널에 URL 표시

- [ ] `railway domain` 실행 후 터미널에 나온 URL(예: `https://xxx.up.railway.app`) 확인
- [ ] 해당 URL로 접속해 앱 동작 확인
