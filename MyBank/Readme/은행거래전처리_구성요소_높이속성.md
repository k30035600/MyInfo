# 은행거래 전처리 - 전처리 영역 구성요소 및 높이 속성

은행거래 전처리 페이지에서 **전처리전/전처리후** 영역의 구성요소와 높이·패딩 관련 속성을 정리한 문서입니다.

---

## 1. HTML 구조 (구성요소)

```
.preprocess-container
├── .wrapper-box
│   ├── .box.left-box          ← 전처리전 박스
│   │   ├── .box-header
│   │   │   ├── .box-header-row
│   │   │   │   ├── .box-header-left
│   │   │   │   │   ├── .box-title ("전처리전")
│   │   │   │   │   └── #left-workspace-path (작업폴더)
│   │   │   │   └── .box-header-right
│   │   │   │       ├── .stat-item (집계, 입금, 출금)
│   │   │   │       └── #left-file-filter-section (필터)
│   │   └── .box-content#left-content
│   │       └── #source-data-table (전처리전 테이블)
│   └── .box.right-box         ← 전처리후 박스
│       ├── .box-header
│       │   ├── .box-header-row
│       │   │   ├── .box-header-left
│       │   │   │   ├── .box-title ("전처리후")
│       │   │   │   └── #right-file-name (작업파일명)
│       │   │   └── .box-header-right
│       │   │       ├── .stat-item (집계, 입금, 출금)
│       │   │       └── #right-filter-section (년/월/파일 필터)
│       └── .box-content#right-content
│           └── #processed-data-table (전처리후 테이블)
```

## 3. 반응형(미디어 쿼리) 요약

- **max-width: 768px**
  - `.preprocess-container`: min-height 450px, height auto
  - `.box`: min-height 250px, max-height 40vh
  - `.box-header`: padding 6px
  - `.box-header-row` / left·right: padding 0 3px
  - `.box-content`: padding 3px
  - table: font-size 9px, th/td padding 2px 4px, line-height 1.2

- **max-width: 360px**
  - table: font-size 8px, th/td padding 2px 3px, line-height 1.2

## 4. 요약 한 줄

- **전처리 영역 전체**: `.preprocess-container` min-height 450px, flex:1
- **전처리전/전처리후 박스**: flex:1, min-height:0, 768px 이하에서만 min-height 250px, max-height 40vh
- **헤더**: 전처리전/후는 min-height:0, padding 4px 8px
- **테이블 담는 영역**: `.box-content` flex:1, padding 6px 6px 20px 6px, overflow-y auto
- **테이블 셀**: th/td padding 2px 4px, line-height 1.2, 행 opacity 0.7
