# 서버 연결 확인

## 서버 상태
✅ **서버가 정상 작동 중입니다!**
- 포트: 5000
- 상태: LISTENING
- 응답 코드: 200 OK

## 브라우저 접속 방법

### 1. 직접 주소 입력
브라우저 주소창에 다음 중 하나를 입력하세요:
- `http://localhost:5000`
- `http://127.0.0.1:5000`

### 2. 연결이 안 될 경우

#### A. 브라우저 캐시 삭제
1. Ctrl + Shift + Delete
2. 캐시된 이미지 및 파일 삭제
3. 페이지 새로고침 (Ctrl + F5)

#### B. 다른 브라우저로 시도
- Chrome, Edge, Firefox 등 다른 브라우저로 시도

#### C. 방화벽 확인
1. Windows 설정 > 방화벽
2. Python이 허용되어 있는지 확인

#### D. 포트 확인
PowerShell에서 다음 명령 실행:
```powershell
netstat -ano | findstr :5000
```
LISTENING 상태가 보이면 서버가 실행 중입니다.

## 서버 실행 방법

### 방법 1: 배치 파일 실행
```
run_server_utf8.bat
```
또는
```
start_flask.bat
```

### 방법 2: 직접 실행
```
python app.py
```
