## GitHub Actions로 서버에서 실행

프로젝트에 **Actions** 탭이 있다면, 설정된 워크플로우를 선택하여 **Run workflow**를 누르면 GitHub 서버에서 바로 코드를 구동할 수 있습니다.

- **Actions** → **Run workflow** (왼쪽) → **Run workflow** 버튼
- 워크플로우: `run-workflow.yml` (체크아웃 → Python 설정 → 의존성 설치 → 실행)
