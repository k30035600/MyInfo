# card_category 기본 규칙 (DEFAULT_CARD_CATEGORY_ROWS)

## 개요

- **분류**: 업종분류(업종코드), 계정과목(지출 항목)
- **키워드**: 가맹점명에 포함되면 해당 행 규칙 적용. `/`로 여러 키워드(OR) 지정.
- **카테고리**: 매칭 시 부여되는 라벨.

위 규칙으로 `create_category_table()`이 card_category.xlsx를 만들고, `apply_category_from_merchant()`가 가맹점명에 키워드를 적용해 카테고리를 채웁니다.

**정렬**: 계정과목(카테고리) 기준 가나다순. 선매칭(파리바게뜨/베이커리, 씨유, (주)이마트/롯데마트/식자재)은 병원·기타잡비(마트)보다 먼저 적용.
